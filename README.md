# Répertoire pour les exercices de Mobile 2026

Ce projet va vous servir de dépôt git pour les exercices du cours de mobile 2026.
Soyez bien attentifs à respecter précisément les instructions dans les fiches, particulièrement côté nommage - des tests automatiques sont prévus pour valider vos résultats dans le cadre de l'évaluation continue (comptant pour 20% des points).

Pour lancer vous mêmes les tests:

- Exécuter le fichier copy_test.dart:

````bash
dart copy_test.dart
````

Ceci va copier les tests dans les folters tuto1, tuto2, etc que vous aurez créer.

Une fois ceci fait, vous pouvez lancer les tests d'une fiche précise avec:

````bash
flutter test
````
